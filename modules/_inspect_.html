Const inspect

inspect: (Anonymous function) = inspectWithPreamble('')
  • Defined in inspect.ts:92

Returns type as a string from TS expression. Creates an inline TS snippet and parses type from the provided input.

example
inspect('Record<keyof any, Array<number>>'); // "{ [x: string]: number[]; [x: number]: number[]; };"
example
inspect({a: 'string | number', b: 'boolean'}); // {a: "string | number", b: "boolean"}
